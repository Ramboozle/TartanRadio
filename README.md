# Tartan Radio System v2.1

Welcome to the Tartan Radio project. This repository contains the source code for both the central management server and the Raspberry Pi/Windows audio clients.

## Project Structure
- **/server**: Node.js backend with Express, Socket.io, and SQLite.
- **/newclient**: Node.js client logic for playback and synchronization (supports Pi & Windows).
- **/Docs**: Comprehensive documentation and usage guides.

## Documentation
Please refer to the following files in the `Docs/` directory for detailed information:

1. [**README.md**](./Docs/README.md) - Project overview, features, and setup instructions.
2. [**GUIDE.md**](./Docs/GUIDE.md) - Detailed usage guide for the Admin Dashboard and Local Client UI.
3. [**API.md**](./Docs/API.md) - Technical documentation for API endpoints and Socket.io events.
